HarrisonLand Version 1.0.1
This is the second version of the game.

New things:
	-Placeable blocks
	-Higher but not high enough FPS
	-Mouse cursor
	-In game terrain re-generation

Notes:
	The game is pretty laggy and runs at a bad 13-19 FPS on my laptop, probably not doing
	much better on other computers.

-Controls:
	ESC - Exit game
	F1 - Toggle debug mode
	F2 - Regenerate terrain
	W - Jump
	A - Move left
	D - Move right
	Q - Move inventory selection left
	E - Move inventory selection right
	S - Save world
	R - Recenter mouse on screen
	ENTER - Increase center height offset
	RSHIFT - Decrease center height offset
	LMOUSEBUTTON - Place block
	RMOUSEBUTTONN - Destroy block/Hit
	MOUSEWHEELSCROLL - Move through inventory

To run the script located in this directory, place it in the root directory of this game. It
will not work otherwise.
